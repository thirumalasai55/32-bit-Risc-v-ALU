# 32-bit-Risc-v-ALU
This project successfully verified a 32-bit RISC-V ALU, hitting 100.0% functional coverage. It used an Object-Oriented SystemVerilog testbench with a Golden Reference Model. By applying Constrained-Random Verification (CRV), it rigorously tested physical hardware limits, max shift boundaries, and extreme edge cases across ten thousand test cycles.
